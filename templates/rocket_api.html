Этот сервис позволяет получать некачественные картинки жабок. Для того, чтобы получить картинку жабки, нужно сформировать правильный запрос.

Пример запроса: https://abrakadabra.pythonanywhere.com/rocket/api?apikey=DEMO_KEY

API принимает следующие параметры:

  • count — число от 1 до 254, количество картинок

Ваша цель

  • Сформировать такой запрос, чтобы получить одну картинку (не в формате JSON)
  • Сформировать такой запрос, чтобы получить 20 картинок в формате JSON
